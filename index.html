HTML4 vs HTML5
Features HTML4 HTML5
New Elements and Semantic Tags Elements like nav, header were not present. New element for web structure like nav, header, footer etc.
Attributes Attributes like charset, async and ping are absent in HTML. Attributes of charset, async and ping are a part of HTML 5.
Audio and Video Support Required plugins (e.g., Flash) for multimedia Native support for audio and video elements
Form Enhancements Limited input types (e.g., text, password) Added new input types (e.g., date, email, number, etc.)
